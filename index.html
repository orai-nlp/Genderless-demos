Hizlari-bektore manipulazioaren bidezko genero-anbiguoko hizketaren sintesia euskaraz

Autoreak: Xabier Sarasola, Ander Corral, Igor Leturia, IÃ±igo Morcillo

Laburpena: Gero eta interes handiagoa dago genero-anbiguko ahotsa duten text-to-speech (TTS) sistemetan, besteak beste, laguntzaile birtualetan eta bozgorailu adimendunetan genero-alborapenak eta estereotipoak saihesteko duten ahalmenagatik. Artikulu honetan, ahots-bihurketa teknika berriak aplikatzen dizkiegu ahots-bektoreei sare neuronaletan oinarritutako genero-anbiguoko euskarazko TTS sistemak lortzeko. Hizlari-bektoreak hiztun anitzeko Tacotron 2-a entrenatuz lortzen dira. Hizlari-bektoreen normalizazioa eta eskala parametro bat erabiltzen duten eta erabiltzen ez duten sistemak konparatu ditugu, baita genero bakoitzaren batez besteko bektore eta ahots errealen bektoreetan erabilera sistema horietan. Emaitzek frogatzen dute aurkeztutako metodoak genero-anbiguoko ahotsak lortzeko baliozkoak direla kalitate onargarriak, baina hobetu daitezkenak, lortuz.

Sistemen arteko konparaketa

Audio bloke bakoitzean esaldi beraren sintesia entzun daiteke hainbat hizlari eta sistemaren konbinazioarekin. Grabazio originalen zutabean (GT) ausazko grabazio bat aukeratu da entrenamendu datubasetik. Akronimoen esanahia ondorengoa da:

  • GT -> Grabzio originalak
  • MT2 -> Tacotron 2 hizlari-anitza
  • MT2-DLN -> Deep Length Normalization daukan Tacotron 2 hizlari-anitza
  • MT2+GA -> Genero anbiguotasun manipulazioa duen Tacotron 2 hizlari-anitza
  • MT2-DLN+GA -> Deep Length Normalization eta genero-anbiguoko manipulazioa duen Tacotron 2 hizlari-anitza
  • G(FE) -> Frekuentzia nagusi ertaineko gizona
  • G(FA) -> Frekuentzia nagusi altuko gizona
  • E(FB) -> Frekuentzia nagusi baxuko emakumea
  • E(FE) -> Frekuentzia nagusi ertaineko emakumea
  • E(FA) -> Frekuentzia nagusi altuko emakumea
  • C -> Genero-anbiguoko zentroidea
